# Group, Page, and Sound Button

This page explains navigation through groups and pages, and how sound buttons behave.

![Group, Page, and Sound Button](images/group_pages_soundbutton.png)

## Groups and Pages

- Groups: `A` to `J`.
- Pages: 18 pages per group.
- Each page contains 48 sound buttons (6 x 8 grid).

Use the group and page controls to move quickly through your set structure.

## Sound Button States

A sound button can be:

- Empty: no media assigned.
- Assigned: media loaded and ready.
- Playing/played: active playback status.
- Locked/marked/highlighted: control and visual states.
- Missing: referenced media file is not available.
